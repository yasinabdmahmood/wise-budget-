### 2. DATABASE SCHEMA (SQLite)
Generate the database initialization script (`init-db.js`) creating the following tables with proper foreign keys, constraints, and indexes for fast mobile querying:

1. `accounts`
   - `id`: TEXT PRIMARY KEY (UUID)
   - `name`: TEXT NOT NULL (e.g., 'Cash', 'Checking')
   - `currency`: TEXT NOT NULL (e.g., 'USD', 'IQD', 'EUR')
   - `initial_balance`: REAL DEFAULT 0.0
   - `created_at`: DATETIME DEFAULT CURRENT_TIMESTAMP

2. `transactions`
   - `id`: TEXT PRIMARY KEY (UUID)
   - `account_id`: TEXT FOREIGN KEY REFERENCES accounts(id) ON DELETE CASCADE
   - `destination_account_id`: TEXT NULL (Only used if type is 'Transfer')
   - `type`: TEXT NOT NULL (Check constraint: 'Income', 'Expense', 'Transfer')
   - `amount`: REAL NOT NULL
   - `category`: TEXT NOT NULL
   - `description`: TEXT NULL
   - `transaction_date`: DATETIME NOT NULL
   - `created_at`: DATETIME DEFAULT CURRENT_TIMESTAMP

3. `budgets`
   - `id`: TEXT PRIMARY KEY (UUID)
   - `category`: TEXT NOT NULL (or 'All' for overall monthly budget)
   - `amount_limit`: REAL NOT NULL
   - `month_year`: TEXT NOT NULL (Format: 'YYYY-MM')

4. `savings_goals`
   - `id`: TEXT PRIMARY KEY (UUID)
   - `name`: TEXT NOT NULL (e.g., 'Vacation Fund')
   - `target_amount`: REAL NOT NULL
   - `current_amount`: REAL DEFAULT 0.0
   - `target_date`: DATETIME NULL

5. `debts`
   - `id`: TEXT PRIMARY KEY (UUID)
   - `contact_name`: TEXT NOT NULL
   - `type`: TEXT NOT NULL (Check constraint: 'Borrowed', 'Lent')
   - `total_amount`: REAL NOT NULL
   - `remaining_amount`: REAL NOT NULL
   - `title`: TEXT NOT NULL
   - `due_date`: DATETIME NULL
   - `created_at`: DATETIME DEFAULT CURRENT_TIMESTAMP
